## Use Cases

1. Record RTI Request
- Store RTI reference number and date
- Link to concerned Panchayat office

2. Upload RTI Response
- Store reply text or document
- Record reply date

3. RTI Analysis
- Analyst marks response as:
  - Complete
  - Delayed
  - Vague
  - Information Denied

4. View Reports
- Count delayed responses
- Identify repeat evasion patterns
