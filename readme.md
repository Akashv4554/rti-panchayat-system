# RTI Grama Panchayat Transparency System

A mini project built using Django.

## Objective
To improve transparency and accountability at the Grama Panchayat level by
digitally organizing RTI requests, responses, and detecting evasion patterns.

## Scope
- Single Grama Panchayat
- Manual RTI response analysis by an RTI Analyst
- Focus on RTI replies and grievance tracking
